# Sample results

A curated sample of pipeline outputs, illustrating what SPICEM produces.
This is a small selection for demonstration purposes, not the full cohort
result set.

- `example_patient_HK2529/` — full single-patient output (spatial maps,
  coupling heatmap, interaction network, consensus streamlines, flux
  balance) for one representative patient.
- `group_comparisons/` — cohort-level differential comparisons between
  clinical groups (e.g. DKD vs Control).
- `clinical_statistics/` — eGFR / fibrosis / comorbidity correlation
  statistics and summary plots.
- `cohort_consensus_examples/` — a sample of cohort-wide consensus
  streamline plots (top metabolites by detection consistency).
- `merged_streamline_examples/` — a sample of multi-panel combined
  streamline figures.
- `merged_clinical_examples/` — a sample of the strongest eGFR/fibrosis
  correlated metabolite streamline plots (`rho` in filename), plus the
  full `clinical_correlations.csv` summary table.
